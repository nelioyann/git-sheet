Cheat Sheet

Git Commands

Starting with a local file and turn it in a git repository

Turns a folder into a git repository

git init

Check the status of a folder

git status

Commiting the changes made to a new file to a git repository/ Putting the files to commit in the staging area

git add [file] git add .

Removing a file from the staging area

git rm --cached [file]

Comitting files from the staging area

git commit -a[=all] -m[message to add] "actual message"

Adding a remote

git remote add origin[name chosen by convention/can be changed] giturl

Removing a remote

git remote remove origin[or chosen name]

Specifing user identity

Setting the username and email adress

git config --global user.name "ne*****n" git config --global user.email "n*******@****.com"

Checking it

git config --list

Checking the logs/ Last commits made

git log

Quitting Vim

:q

Push on Github. the remote

Shows all the remotes associated with the account

git remote git remote -v

Pushing to master

git push origin master

Pull from github

Pull

git pull origin master
